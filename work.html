Script & Webapp Python Parser ICS

Pour mon utilisation personnelle, je récupérais un ensemble d'événements sous un fichier au format .ics (événement de type calendrier).Mais seulements certains événements m'interessaient. J'ai donc créer un script qui récupérer le fichier depuis une source web, puis qui supprimer le contenu inutile et générer un nouveau fichier, lui aussi étant en ligne.

J'ai peu ainsi appliquer mes connaissances Python et web pour faire une webapp avec l'aide de GoogleAppEngine.
